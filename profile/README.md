# TED Protocol

> Cross-chain Stablecoin FX Exchange with Ultra-Low Slippage

TED Protocol is a decentralized platform for swapping stablecoins across different currencies and blockchains. Unlike traditional DEXs that treat stablecoins as generic tokens, TED Protocol approaches them as real currencies with FX rates, enabling optimized routing and better pricing.

## Features

| Feature | Description |
|---------|-------------|
| **Multi-DEX Aggregation** | Parallel quotes from 5+ DEXs (Curve, Uniswap, PancakeSwap) for best prices |
| **Cross-Chain Bridges** | Circle CCTP, Wormhole, LayerZero integration |
| **Ultra-Low Slippage** | < 0.15% on stablecoin swaps |
| **Non-Custodial** | User-signed transactions, your keys = your funds |
| **Multi-Network** | Ethereum, Arbitrum, Base, BSC, Kaia |
